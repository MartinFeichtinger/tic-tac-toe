# Tic-Tac-Toe
This project originated in 2022 as a soldering kit for children, designed for a summer workshop organized by the OTELO Gmunden. The workshop was spearheaded by Friedrich Feichtinger, who not only organized the event but also developed and designed the electronics for the Tic-Tac-Toe game.

My role in the project was focused on the development of the software. Initially, I created a bot to provide players with a challenging opponent. In the second phase, I adapted the code to run on a microcontroller, which, as you can imagine, resulted in a rather chaotic and disorganized codebase.

Recendly, I attempted to update a small section of the code but quickly realized I had no clear understanding of where to look or how to implement the changes. This was a pivotal moment for me when I recognized that combining a game bot and GPIO multiplexing logic into a single file was far from an ideal approach.

Motivated to improve my coding practices, I decided to refactor the project and reorganize it into a cleaner, more maintainable structure, ultimately enhancing both my coding skills and the quality of the project.
